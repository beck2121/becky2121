Objetivos de una Página Web

1Objetivos General

Diseñar páginas web haciendo uso de todos los conocimientos dados en el desarrollo de todas las clases, utilizando herramientas que modifiquen el tipo de letra, color de fondo, imágenes y videos en una página web.

2Objetivo Especifico I

Crear páginas web haciendo uso de herramientas como HTML y CSS3, aprendiendo ampliamente del diverso mundo de la programación.

3Objetivo Especifico II

Reconocer los diversos usos y habilidades que se pueden adquirir con el lenguaje de HTML aplicado en la creación de páginas web.
